# Plugin playground

This page is driven by the shared Markdown pipeline in `astro.config.mjs`.

## Contents

## What is enabled

- `remark-toc` inserts a table of contents under the `Contents` heading.
- `rehype-slug` adds stable IDs to headings.
- `rehype-autolink-headings` appends jump links to headings.
- `rehype-external-links` marks external links and forces safe external-link attributes.

Hover or focus the heading area after the page loads. The `#` markers are injected by the rehype plugin, not written into the source Markdown.

## Why it matters

These plugins are low-friction improvements for a static documentation site because they work across all Markdown entries without changing each page template.
